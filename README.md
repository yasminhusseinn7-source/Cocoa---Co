# 🍫 Cocoa & Co. Pâtisserie

A modern and elegant bakery & dessert website built using HTML, CSS, and JavaScript.

## 🌐 Live Demo

🔗 https://yasminhusseinn7-source.github.io/Cocoa---Co/

---

## 📖 About the Project

Cocoa & Co. is a responsive website designed for a premium pâtisserie.  
The website provides visitors with a beautiful experience to explore the menu, learn about the bakery, find the location, and contact the team.

---

## ✨ Features

- Responsive Design
- Elegant Modern UI
- Home Page
- Menu Page
- About Section
- Contact Section
- Google Maps Integration
- Contact Form with JavaScript Validation
- Interactive Buttons & Hover Effects
- Clean and Organized Layout
